# Veterinary Patient Manager

Patient management app for veterinary clinics built with React and TypeScript.

<img width="1920" alt="screenshot" src="./public/screenshot.png" />

## Features

- **Patient CRUD**: Add, edit and delete patients with pet, owner and symptom data.
- **Local persistence**: Data is automatically saved to localStorage.
- **Form validation**: Real-time validation with react-hook-form.
- **Notifications**: Visual feedback with toast notifications.
- **Responsive design**: Adapted for mobile and desktop.

## Tech Stack

- React 19
- TypeScript 6
- Tailwind CSS v4
- Zustand
- react-hook-form
- react-toastify
- Vite 8

## Installation

```bash
npm install
npm run dev
```

## Scripts

| Command | Description |
|---|---|
| `npm run dev` | Start development server |
| `npm run build` | Build for production |
