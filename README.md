# Veterinary Patient Manager

Patient management app for veterinary clinics built with React and TypeScript.

<img width="1920" height="994" alt="screencapture" src="https://github.com/user-attachments/assets/bd49e829-a1d1-432a-ac20-0eab38290072" />

## Features

- **Patient CRUD**: Add, edit and delete patients with pet, owner and symptom data.
- **Local persistence**: Data is automatically saved to localStorage.
- **Form validation**: Real-time validation with react-hook-form.
- **Notifications**: Visual feedback with toast notifications.
- **Responsive design**: Adapted for mobile and desktop.

## Tech Stack

- React 19
- TypeScript 6
- Tailwind CSS v4
- Zustand
- react-hook-form
- react-toastify
- Vite 8

## Installation

```bash
npm install
npm run dev
```

## Scripts

| Command | Description |
|---|---|
| `npm run dev` | Start development server |
| `npm run build` | Build for production |
| `npm run preview` | Preview production build |
| `npm run lint` | Run linter |
